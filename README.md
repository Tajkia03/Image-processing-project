Image Processing Project

This project was developed as part of the CSE 428 (Digital Image Processing) course.

Features

Image filtering (smoothing, sharpening)
Image transformation
Edge detection
Basic image analysis
Technologies Used

Python
NumPy
OpenCV / PIL
Matplotlib
How to Run

Install required libraries: pip install numpy opencv-python matplotlib
Open the notebook: jupyter notebook
Run all cells
Purpose This project demonstrates fundamental image processing techniques and their practical implementation.
